Tutorial: Umgang mit Komponenten in NanoPLM

Einleitung

In diesem Tutorial lernen Sie, wie Sie als Maschinenbauingenieur effektiv mit Komponenten in der PLM-Lösung NanoPLM arbeiten. NanoPLM hilft Ihnen dabei, Konstruktionsdaten zu verwalten, Versionen zu kontrollieren und die Zusammenarbeit im Team zu verbessern. Dieses Tutorial führt Sie Schritt für Schritt durch die grundlegenden Prozesse.

Schritt 1: Anmeldung in NanoPLM

Bevor Sie mit NanoPLM arbeiten können, müssen Sie sich mit Ihren Zugangsdaten anmelden. Folgen Sie diesen Schritten:

  1. Öffnen Sie die NanoPLM-Anwendung oder -Webseite.
  2. Geben Sie Ihre Benutzerkennung und Ihr Passwort ein.
  3. Klicken Sie auf Anmelden, um Zugang zu erhalten.

Schritt 2: Komponenten suchen und anzeigen

Um eine Komponente in NanoPLM zu finden, verwenden Sie die Suchfunktion:

  1. Gehen Sie zum Suchfeld auf der Startseite oder im Hauptmenü.
  2. Geben Sie den Namen, die ID oder einen anderen Identifikator der Komponente ein.
  3. Klicken Sie auf Suchen, um die Ergebnisse anzuzeigen.
  4. Wählen Sie die gewünschte Komponente aus, um Details wie technische Spezifikationen, Versionen und Status anzuzeigen.

Schritt 3: Eine neue Komponente erstellen

Wenn Sie eine neue Komponente erstellen möchten, gehen Sie wie folgt vor:

  1. Klicken Sie im Hauptmenü auf Neue Komponente erstellen.
  2. Füllen Sie das Formular aus, indem Sie wichtige Informationen wie Name, Beschreibung, Material und Maße angeben.
  3. Hängen Sie bei Bedarf Dokumente oder CAD-Dateien an.
  4. Klicken Sie auf Speichern, um die Komponente in der Datenbank zu sichern.

Schritt 4: Änderungen an einer Komponente vornehmen

Um eine vorhandene Komponente zu bearbeiten:

  1. Öffnen Sie die Komponente durch die Suche oder das Navigieren im Menü.
  2. Klicken Sie auf Bearbeiten.
  3. Nehmen Sie die gewünschten Änderungen vor, z. B. Aktualisierung der Maße oder Hinzufügen von Kommentaren.
  4. Speichern Sie die Änderungen und fügen Sie, falls erforderlich, eine Versionsbeschreibung hinzu.

Schritt 5: Zusammenarbeit und Freigabe

Die Zusammenarbeit in NanoPLM ist ein zentraler Bestandteil. So geben Sie eine Komponente frei:

  1. Öffnen Sie die gewünschte Komponente.
  2. Klicken Sie auf Freigeben und wählen Sie die entsprechenden Teammitglieder aus.
  3. Optional können Sie Zugriffsrechte festlegen, z. B. nur Leserechte oder Bearbeitungsrechte.
  4. Fügen Sie bei Bedarf eine Nachricht hinzu und klicken Sie auf Senden.

Zusammenfassung

In diesem Tutorial haben Sie die grundlegenden Schritte für den Umgang mit Komponenten in NanoPLM gelernt: Anmeldung, Suche, Erstellung, Bearbeitung und Freigabe. Diese Prozesse helfen Ihnen, effizienter zu arbeiten und Ihre Konstruktionen besser zu verwalten. Nutzen Sie die umfangreichen Funktionen von NanoPLM, um Ihre Projekte erfolgreich umzusetzen!
